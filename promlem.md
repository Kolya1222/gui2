**Таблица 5 – Проблемы бизнес-процесса предпродажного взаимодействия**

| № | Проблема | Причина | Степень значимости |
|:--:|:--|:--|:--|
| 1 | Ручное распределение входящих заявок руководителем | Отсутствие в CRM-системе настроенных правил автоматического распределения заявок по тематике запроса | Высокая |
| 2 | Отсутствие региональных цен на сайте | Архитектурное ограничение CMS MODX Revolution – невозможность привязки разных цен к одному товару для разных поддоменов | Высокая |
| 3 | Менеджер вынужден переходить на сайт для уточнения информации о товаре | Отсутствие интеграции между CRM и 1С, недостаточная информативность заявки, поступающей из формы обратной связи | Высокая |
| 4 | Формирование КП в 1С с последующей отправкой через Outlook | Разрозненность систем (CRM, 1С, почтовый клиент), отсутствие единого рабочего пространства менеджера | Высокая |
| 5 | Ручное изменение статуса сделки в CRM | Отсутствие автоматической смены статуса при получении подтверждения от клиента | Высокая |
| 6 | Дублирование разделов и страниц сайта | Отсутствие единой политики управления контентом, развитие сайта разными специалистами без координации | Высокая |
| 7 | Падение объема переходов из поисковых систем | Наличие дубликатов страниц, ошибки в микроразметке, малый объем проиндексированных страниц | Высокая |
| 8 | Неоптимальная глубина сайта (8 уровней) | Отсутствие аудита структуры сайта при его развитии | Средняя |
| 9 | Неоптимальная структура страниц (текст до призыва к действию) | Устаревший подход к представлению информации, ориентированный на SEO, а не на пользователя | Средняя |
| 10 | Текстовые элементы со слабой визуальной разбивкой ("простыни" текста) | Отсутствие современных стандартов верстки и представления контента | Средняя |
| 11 | Малый объем страниц, приводящих на сайт | Слабые заголовки и meta-описания в связке с низким качеством контента | Средняя |
| 12 | Высокий процент отказов на небрендовых запросах | Комплексная проблема: дубликаты, скорость, качество контента, отсутствие микроразметки | Средняя |
| 13 | Отсутствие фильтрации в больших разделах | Функциональное ограничение CMS, приводящее к созданию избыточных подкатегорий | Средняя |
| 14 | Переодически скорость работы сайта превышает норму | Тяжелые изображения, неоптимизированный код, ограничения хостинга | Средняя |
| 15 | Нерелевантный поиск по сайту | Отсутствие настройки поискового движка или использование стандартного, не учитывающего специфику каталога | Средняя |
| 16 | Ошибки в микроразметке | Наличие в названиях товаров и пунктах меню кавычек, препятствующих корректной разметке | Средняя |
| 17 | Множество пустых страниц без наполнения | Страницы закрыты для входа с сайта, но индексируются поисковиками | Средняя |
| 18 | Большой объем однотипных шаблонов (56 шт.) | Отсутствие унификации при разработке, копирование шаблонов вместо создания гибких | Низкая |
| 19 | Сложность поиска места изменения на странице | Множество мест хранения данных (шаблоны, чанки, сниппеты, TV-поля, файловая система) | Низкая |
| 20 | Низкая скорость работы административной панели | Устаревшая архитектура CMS, использование ExtJS 3.4 (2011 г.) | Низкая |
| 21 | Панель управления не сохраняет текущий раздел | Недоработка интерфейса административной панели | Низкая |
| 22 | Сложность разработки собственных модулей | Устаревшая документация (2011 года), отсутствие современных практик разработки | Низкая |
| 23 | Большинство дополнений платные | Модель распространения дополнений в экосистеме MODX | Низкая |
| 24 | Низкая отказоустойчивость при изменении типов ресурсов | Ошибки в платных модулях при изменении базовых настроек | Низкая |
| 25 | Сложность отлова ошибок | Ошибки фиксируются в общий лог без разграничения по типам и критичности | Низкая |
| 26 | Архитектура CMS не приспособлена под Headless-подход | Технологические ограничения, невозможность использовать сайт как единый источник данных для разных платформ | Низкая |
| 27 | Сложность масштабирования | Необходимость учитывать требования различных дополнений при изменениях | Низкая |
| 28 | Использование нескольких CMS (MODX Evolution, MODX Revolution, Evolution CMS) | Отсутствие стратегии развития, накопление технического долга | Низкая |
